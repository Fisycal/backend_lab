# Backend Concepts Lab

## Project goal
Explain that this repo is for learning backend fundamentals through hands-on

## Milestone 1 scope
List:
- What is an API
- REST basics
- HTTP methods
- HTTP status codes

## Tech stack
- Python
- FastAPI
-Uvicorn

## Implemented endpoints
- GET /
- GET /users/
- GET /users/{user_id}
- POST /users/
- PUT /users/{user_id}
- PATCH /users/{user_id}
- DELETE /users/{user_id}

## How to run
Include:
1. Create venv
2. Install requirements
3. Run uvicorn

## What I learned
In this lab, I learnt what API is. How to create an app using Flask module, implements HTTP methods, better understanding of status code. I created a virtual environment, install needed required packages, and ran the main app using uvicorn which reloads every time I make changes to my codes.
